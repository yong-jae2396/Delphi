# Delphi Component

****

### 구성요소 또는 Component

* 다시 사용할 수 있는 범용성을 위해 개발된 소프트웨어 구성 요소

****

### Component 기반 개발(CBD)

컴포넌트 기반 개발(Component - Based - Development **CBD**)은 기존의 시스템이나 

소프트웨어를 구성하는 Component를 조립해서 하나의 새로운 응용 프로그램을 

만드는 소프트웨어 개발방법론이다.

****
